# VisiTrack

VisiTrack is a web- based system designed to manages and track students attendance using modern technologies.

## Features:-
- User Login/ Signup
- OTP Verification
- Attendance Tracking
- Clean UI Design
- Pythob based backend testing modules

## Build With:-
- HTML 5
- CSS 3
- Python 3.x
- Flask
- Git and GitHub
